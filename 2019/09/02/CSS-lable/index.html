    CSS规则如何级联

    优先级：

    （1）就近原则
    如果两个选择器完全相同，那么后出现的选择器优先级较高。

    （2）具体性原则

    如果一个选择器比其他选择器更加具体，那么具体的选择器优先于一般的选择器

    （3）重要性

    你可以在任意属性值后面添加!important来强调这个规则比应用于同一元素的其他规则更重要
