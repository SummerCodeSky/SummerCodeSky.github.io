flex布局

​ 小程序推荐使用flex布局，也可以使用其他布局（Div、CSS等）

flex-direction flex方向布局

flex-direction：row

默认从左向右排列

flex-direction:row-reverse

默认从右向左排列

flex-direction:column

从上向下排列

flex-direction:column-reverse

从下向上排列

flex-wrap flex换行

flex-wrap：nowrap

默认不换行

flex-wrap：wrap

换行

flex-wrap：wrap-reverse

逆向换行

justify-content flex对齐方式与间距

justify-content：flex-start

默认左对齐

justify-content:flex-end

右对齐

justify-content:center

居中对齐

justify-content:space-around

在成员元素周围包裹空白

justify-content:space-between

在成员元素中间留空白

order字段

通过数字对flex内部容器成员设置显示顺序

order：数字

eg: order：1；order:3;order:2;

按照1，2，3的顺序显示

flex字段

配置成员行级属性所占比例

flex：数字

eg: flex:3;flex:2;flex:4

分成9份，占不同宽度

基础组件

通用标签

id

class

style

hidden(没有懒加载)

data-*

bind * /catch *适用于EventHandler

组件7大类

视图组件

view组件

hover-class=”none”

指定按下去的样式

hover-stop-propagation=”false”

指定是否组织本节点的祖先节点出现点击态

hover-start-time=“2000”

按住后多久出现点击态

hover-stay-time=“1000”

手指松开后点击保留时间

scroll-view组件

< scroll-view>< /scroll-view>

上下滑动/左右滑动

scroll-x=”true”

scroll-y=”false”

bindscrolltoupper=”scrolltoupper”

有一个距离顶部还有多少的时候触发这个事件

< upper-threshold =”0”> //默认为50

注意：scrolltoupper只是事件名称

bindscrolltoupper=”scrolltolower”

有一个距离底部还有多少的时候触发这个事件

< lower-threshold =”0”> //默认为50

注意：scrolltolower只是事件名称

scroll-top=”100”

竖向滚动条位置

scroll-left=”100”

横向滚动条位置(距离左边多远)

enable-back-to-top=”true”

支持滚动条从底部到顶部

scroll-with-animation=”true”

滚动时动画过度

bindscroll=”bindscroll”

注意：bindscroll为时间名

scroll-into-view=”b”

注意：b为id名

值为子元素id 设置那个方向，滚动到哪个方向的位置

swiper组件

即轮播图

movable-area和movable-view

direction=“none”

属性值有：all/vertical/horizontal等

movable-area在movable-view里面可移动的范围

inertia=“true”

有惯性

out-of-bounds=”true”

超越边界区域（即movable-view）是否还可以移动

x=”50”

定义x坐标

y=“50”

定义y坐标

damping=”100”

阻尼系数越大移动的越快

friction=”100”

摩擦系数越大，滑块停止越快

disable=”true”

true为禁用

scale=”true”

是否支持双指缩放

scale-max

缩放倍数最大值

scale-min

缩放倍数最小值

scale-value

缩放倍数取值（范围在0.5-10之间）

bindchange=”事件名”

滑动期间的事件

bindscale=”事件名”

缩放时触发的事件

icon组件

type=“success” ；success-nocircle/info/warn/waiting/cancel/download/search/clear等

size=“12”

color=“blue”

text文本组件

selectedable=”true”

可以被复制

rich-text富文本组件

progress进度条组件

percent=”30”

以百分比的形式显示

stroke-width=“”

进度条宽度

activeColor=“red”

已进行进度条长度

backgroundColor=“blue”

active=”true”

选择进度条播放动画

active-mode=”forwards”

从上次结束位置开始

表单组件

button按钮

size=“string” mini等

type=”string” default/primary/warn等

plain=”｛｛true｝｝”

disabled=”｛｛true｝｝”

loading=”｛｛true｝｝”

form-type=“string” submit/reset

open-type=”string”

hover-class=”string”

checkbox label

bindingchange

两个连用

< label for=”checkbox 的id”>< /label>

input

cursor-spacing=”100”

光标与输入框的距离

focus=｛｛true｝｝

自动唤醒键盘并聚焦

confirm-type

设置键盘右下角文字（改变“完成”为“下一项”）

confirm-hold=“｛｛true｝｝”

点击下一项，页面并不会进行什么操作

cursor

光标所在位置

select-start

select-end

默认打开光标的起始位置

adjust-position=“｛｛false｝｝”

键盘弹起时是否向上推动页面

bindinput触发的事件

点击输入框输入的时候

bindfocus

获取焦点事件

bindblur

失去焦点事件

bindconfirm

点击下一项的时候触发的事件

picker –选择器

普通选择器–model=selector

range=”“ 设置范围

range-key=”数组的关键字名”

bindchange=“绑定事件名”

cancel=“触发取消事件名”

columnchange

多列选择器 –muliselector

picker-view

bindchange=“绑定事件名”

radio

bindchange=“绑定事件名”

checked

disable

color

slider –滑动选择器

min

max

show-value

step –步长

backgroundColor

block-color=“green”
